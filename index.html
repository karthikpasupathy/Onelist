      Help & Shortcuts

      Editing

      • Enter New bullet line (- )
      • Shift + Enter Plain newline
      • /today Insert today's date (dd-MM-yyyy)
      • /tomorrow Insert tomorrow's date
      • /time Insert current time
      • /line Insert a divider line
      • /name Expand a saved snippet — type / for suggestions

      Undo

      • Ctrl + Z Undo last change

      Keyboard shortcuts

      • Ctrl + Shift + D Insert today's date at cursor
      • Ctrl + K Open search
      • Ctrl + F Open search
      • Ctrl + / Open this help
      • Esc Close open modal

      On Mac, use Cmd instead of Ctrl.

      Search

      • Type any text to find matching lines
      • Type #tag to filter lines by hashtag (e.g. #work)
      • Switch scope to search all years or just the current year

      Add New Year

      Enter exactly 4 numbers to create a new year note.
